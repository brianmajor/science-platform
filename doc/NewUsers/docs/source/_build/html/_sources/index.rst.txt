Welcome to the documentation for CANFAR's Science Portal!
==================================================================

The CANFAR Science Portal 
provides astronomers the computing resources they
need to work with their data.  In addition to the documentation
here, aimed at newer users, more advanced topics are covered
at https://github.com/opencadc/science-platform/tree/master/doc
